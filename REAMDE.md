## Installation

1. Install all dependencies on the front-end and back-end
```
npm install
cd client
npm install
```
2. Start the back-end/Express server

```
npm start
or
npm run dev (nodemon)
```
3. Start the front-end/Webpack dev server

```
cd client
npm run dev
```

4. Build the front-end production files

```
cd client
npm run build
```
5. Visit http://localhost:3000
